# Elephant Bot

kupi slona - это простейший Telegram-бот, который отвечает на сообщения определенного пользователя фразой "все так говорят, а ты купи слона", подключенный со стороны аккаунта.

## Требования

- Python 3.6+
- Библиотека Telethon
